# FHEVM Example: Encrypted Voting

This repo is a demo built on top of Zama's FHEVM hardhat template.

## How to run locally

1. npm install
2. npx hardhat compile
3. npx hardhat test ./examples/encrypted-voting/test/encryptedVoting.test.js

## Notes
- This uses placeholder ciphertext (UTF-8 bytes).
- Replace with Zama SDK ciphertext for real FHE integration.
